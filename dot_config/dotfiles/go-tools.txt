# optional (defaults to @latest if omitted). Lines starting with # are ignored.
#
# Installed automatically by run_onchange_after_10-install-packages.sh.tmpl into
# `$(go env GOBIN)` (or $GOPATH/bin) whenever you `chezmoi apply`. Add a line

# The gitea MCP server. The Claude config launches it with `go run …@latest`,
# which recompiles (~12s) on a cold cache and overruns Claude's MCP startup
# window. Installing it here warms the build cache so `go run` starts instantly.
