# Go tools to install with `go install`. One module path per line; `@version`
# optional (defaults to @latest if omitted). Lines starting with # are ignored.
#
# Installed automatically by run_once_after_10-install-packages.sh into
# `$(go env GOBIN)` (or $GOPATH/bin) whenever you `chezmoi apply`. Add a line
# here and it shows up everywhere — no remembering `go install ...`.
#
# Examples (uncomment or add your own):
# github.com/jesseduffield/lazygit@latest
# mvdan.cc/gofumpt@latest
# github.com/go-delve/delve/cmd/dlv@latest
# github.com/charmbracelet/glow@latest
